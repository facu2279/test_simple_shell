# Usage: 
Recommended: test in this order:
``` sh
$ ./test ./hsh
$ ./test-create ./hsh
$ ./test-rm ./hsh
$ ./test-exit-args ./hsh
$ ./test-separators ./hsh
$ ./test-comments ./hsh
```
